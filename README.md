# Solar System Simulation

A 2D solar system simulation built using Python and Pygame. This project visually demonstrates planetary motion and orbital mechanics using mathematical concepts like sine and cosine to simulate circular and elliptical orbits.

## Overview

This project simulates planets orbiting around a central sun using basic physics and trigonometry. It is designed as a learning project to understand:

- Game loops in Pygame
- Frame updates and animation
- Circular motion using sine and cosine
- Basic simulation structure in Python

## Features

- Real-time animation of planets orbiting a central sun
- Adjustable orbital speed and radius
- Scalable structure for adding more planets
- Smooth frame-based rendering using Pygame
- Modular code structure for easy updates

## Concepts Used

- Python programming fundamentals
- Pygame library
- Trigonometry (sin, cos)
- Game loop architecture
- Coordinate transformation
